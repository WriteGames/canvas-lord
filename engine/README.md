# Canvas Lord

[![npm version](https://badge.fury.io/js/canvas-lord.svg)](https://badge.fury.io/js/canvas-lord)

Browser embeddable, interactive game tutorial engine, used for WriteGames.com.

## Installation

```sh
# choose based on your package manager
npm i canvas-lord
pnpm add canvas-lord
yarn add canvas-lord
```
